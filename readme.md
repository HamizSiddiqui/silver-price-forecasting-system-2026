# Silver Price Forecasting System

This project predicts silver prices using Prophet and automates retraining using GitHub Actions.

## Features
- Daily data scraping
- Automated model retraining
- API deployment
- Time-series forecasting

## Tech Stack
- Python
- Prophet
- FastAPI
- GitHub Actions
- Vercel
